# README - aMostra de Estatística XIX

Site feito por alunos de estatística do ime-usp para a edição de 2026 da aMostra.

## Sobre o Projeto

Este repositório contém o código-fonte do site oficial da **19ª edição da aMostra de Estatística**, um evento acadêmico organizado por estudantes de Estatística da Universidade de São Paulo (USP). O site tem como objetivo centralizar informações sobre o evento, incluindo programação, inscrições, palestrantes, conteúdos institucionais e materiais educativos relacionados à área de Estatística. Para a edição de 2026, o site está passando por uma expansão significativa, com a inclusão de novas seções e melhorias de usabilidade, conteúdo e design.

## Tecnologias Utilizadas

- **HTML5**: Estrutura do site
- **Tailwind CSS**: Framework CSS para estilização responsiva
- **JavaScript**: Interatividade e animações
- **Font Awesome**: Ícones
- **Google Fonts**: Tipografia

## Funcionalidades Principais

1. **Página Inicial**:
   - Apresentação do evento
   - Contador regressivo atualizado
   - Informações principais (data, local, inscrições)

2. **Estatísticas e Edições Passadas**:
   - Dados sobre edições anteriores (participantes, patrocinadores, palestras)
   - Galeria de imagens de edições anteriores

3. **Programação (em atualização)**:
   - Cronograma completo do evento (horários, títulos das palestras, palestrantes, descrições)]
   - Integração com Google Agenda

4. **Palestrantes (em atualização)**
   - Cards individuais com: foto, nome, empresa, mini biografia e linkedin)
   
5. **Inscrições**:
   - Links para formulários de inscrição (a serem ativados)
   - Informações para participantes

5. **Patrocinadores (em atualização)**
   - Exibição de empresas apoiadoras

6. **Equipe**
   - Lista da comissão organizadora
   - Possível inclusão de fotos dos membros
   
7. **Localização**
   - Integração com Google Maps
   - Indicação do local do evento no IME-USP

8. **Blog de Estatística (NOVO-2026)**

   Nova seção criada para ampliar o alcance do evento e divulgar conhecimento: 
   - Aplicações reais de Estatística
   - Conteúdos voltados para estudantes
   - Explicações de temas relevantes da área

5. **Rodapé**:
   - Links para redes sociais
   - Informações de contato
